# 言而有信 (Promise Keeper) 插件

[![版本](https://img.shields.io/badge/version-1.7.2-blue.svg)](https://github.com/xunxiing/astrabot_plugin_promise_keeper)
[![许可](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![兼容性](https://img.shields.io/badge/AstrBot-v3.4+-brightgreen.svg)](https://github.com/AstrBotDevs/AstrBot)

一个为 [AstrBot](https://github.com/AstrBotDevs/AstrBot) 设计的智能插件，旨在解决群聊中“说过就忘”的难题。

你是否也曾遇到朋友或自己在群里立下Flag，说过“马上就去”、“明天一定完成”，结果却悄无声息？**言而有信**插件会自动检测聊天中的承诺性话语，进行记录，并在承诺到期时主动 **@** 用户提醒，甚至还提供中点提醒和有趣的排行榜功能！

## ✨ 功能特性

- **🚀 自动识别承诺**：通过灵活的正则表达式，智能识别聊天中的承诺性话语，包括含有否定词（如“我明天**不**摸鱼”）的承诺。
- **🕒 智能时间解析**：
    - 支持精确时间（如“10分钟后”、“明天下午3点”）。
    - 支持模糊时间（如“马上”、“现在”、“一会”），并可自定义其对应的默认时长。
    - 支持中文数字（如“第二天”、“第三天”）。
- **🔔 双重提醒机制**：
    - **中点提醒**：在承诺时间过半时，发送一次“时间过半”的提醒（短于60秒的承诺除外）。
    - **到期提醒**：在承诺时间截止时，**@** 用户进行最终提醒。
- **🖼️ 丰富的图片指令**：
    - `/言而有信排行`：生成一张精美的图片，展示群内成员的“承诺次数”排行榜。
    - `/言而有信 [@某人]`：生成一张清晰的个人承诺列表图片，方便追溯和管理。
- **⚙️ 高度可配置**：所有识别规则和模糊词的定义都存储在外部 `regex_config.json` 文件中，方便用户自由修改和扩展，无需改动代码。
- **💪 稳定可靠**：解决了跨平台兼容性（Web UI）、重复记录、数据错乱等多个棘手问题，逻辑健壮。

## 🔧 安装与部署

1.  从本仓库下载所有文件。
2.  将整个 `astrabot_plugin_promise_keeper` 文件夹放入您 AstrBot 的 `data/plugins/` 目录下。
3.  确保 `requirements.txt` 文件在插件目录中。AstrBot 在启动时会自动检测并安装所需的库 (`dateparser`)。
4.  重启 AstrBot 或在 Web 管理面板中重载插件。

## ✨ 核心功能

- **智能承诺识别**: 通过高度可配置的正则表达式，自动从日常聊天中识别出包含承诺性的话语。无论是“我明天就做”，还是“我保证一会不摸鱼”，都能轻松捕获。
- **灵活的时间解析**: 不仅能理解“10分钟后”、“第二天”这样的精确时间，还能处理“马上”、“现在”、“回头”等模糊时间词，并为它们设定不同的默认倒计时。
- **双重提醒机制**:
  - **中点提醒**: 在承诺时间过半时，发送一次“时间过半”的友好提醒 (默认对60秒以上的承诺生效)。
  - **到期提醒**: 在承诺时间到达时，主动 `@` 用户进行最终提醒。
- **可视化图片报告**:
  - **排行榜**: 使用 `/言而有信排行` 指令，生成一张精美的承诺次数排行榜图片。
  - **个人列表**: 使用 `/言而有信 [@某人]` 指令，用图片展示自己或指定用户的所有承诺及其状态。
- **高度可配置**: 所有的匹配规则、模糊时间词及其对应的倒计时，都存放在 `regex_config.json` 文件中，方便用户随时调整和扩展，无需修改代码。

## 🚀 如何使用

### 自动识别

本插件的核心功能是全自动的。只要您在聊天中发送了符合预设规则的话，机器人就会自动记录并回复。

> **你**: 我现在就去写作业
